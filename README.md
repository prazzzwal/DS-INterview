# IPL Data Analysis & Machine Learning Project

A comprehensive IPL (Indian Premier League) cricket data analysis project featuring multiple ML models, outlier treatment, feature engineering, and visualizations.

## 📁 Project Structure
├── csv files/ # All dataset files
│ ├── ipl_clean.csv # Cleaned IPL data
│ ├── ipl_features.csv # Engineered features
│ ├── ipl_matches.csv # Raw match data
│ ├── scraped_books.csv # Web scraped data
│ ├── X_train.csv, X_test.csv # Train/test splits
│ └── y_train.csv, y_test.csv # Target variables
│
├── notebooks/ # Jupyter notebooks
│ ├── project_01.ipynb # Main project notebook
│ ├── visualization.ipynb # Data visualizations
│ ├── feature_engineering.ipynb # Feature creation
│ ├── outliers.ipynb # Outlier detection/treatment
│ ├── scaling.ipynb # Feature scaling
│ ├── test_train.ipynb # Train-test split
│ ├── decision_tree.ipynb # Decision Tree model
│ ├── logistic_regression.ipynb # Logistic Regression model
│ ├── k_means.ipynb # K-Means clustering
│ └── qsn1.ipynb # Question 1 analysis
│
├── reports/ # Generated outputs
│ ├── ipl_decision_tree.png # Decision Tree visualization
│ ├── ipl_lr_confusion_matrix.png # Confusion matrix
│ ├── ipl_outliers_before.png # Before outlier treatment
│ ├── ipl_outliers_after.png # After outlier treatment
│ ├── ipl_outliers.png # Outlier analysis
│ ├── selenium_automation.py # Web scraping script
│ └── web_scraping.py # Selenium automation
│
└── README.md
